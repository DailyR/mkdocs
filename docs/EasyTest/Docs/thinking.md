- 四.测试环境的问题
	- 一般客户端的luamon调试需要打开5个东西:
		- 1个是unity，方便查找资源和预设，
		- 1个是exe(有时候可以直接用unity的客户端进行替代)，
		- 1个是开着浏览器的luamon进行调试和看效果，
		- 1个是开着perfectlua的环境去查看客户端代码，
		- 1个是开着sublimetext3进行lua代码的备份和output调试记录。

		- 有时候很unity运行慢就用exe，需要再打开就好
		- 然后svnlog是帮助理解代码的辅助工具，详见一、代码路径
