### 一些独立功能和模块思考

- 一些独立功能，比如文件遍历功能，这些在navigation的工程下面是以函数形式存在，这些大部分是可以直接拿来用的
	
	- 例如在django html 页面展示数据，传值views下的render接口

	- render接口传字典，字典里可以是dict，可以是list或者set,但是首先它是个字典

	- 然后注意点是echart里面的M需要去除，echart只能渲染数字形式的数字或者字符串
